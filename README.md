# Personal Website Base

This is a minimal single-page personal site with sections for:

- intro
- portfolio
- resume highlights
- contact

## Run locally

From this folder in PowerShell:

```powershell
.\start-local.ps1
```

Then open:

`http://127.0.0.1:3000`

## Edit first

Update these placeholders in `index.html`:

- `Your Name`
- bio and intro copy
- project titles and links
- resume entries
- email, LinkedIn, and GitHub links
